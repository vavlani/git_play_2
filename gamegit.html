Welcome to the Git Learning Game! In this game, you'll learn intermediate Git commands and concepts through interactive exercises.

Level 1: Branching

Let's start with branching. Use the command git branch feature-branch to create a new branch called 'feature-branch'.

Level 2: Merging

Great! Now, merge 'feature-branch' into 'main'. Use git checkout main and then git merge feature-branch.
